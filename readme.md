##TicTacToe Game

A TicTacToe game in CoffeeScript using Backbone.js TDD-ed with jasmine-node.

You need to install the necessary node modules as npm:
├── backbone @ 0.5.3
├─┬ jasmine-node @ 1.0.12
│ ├── coffee-script @ 1.1.2
│ └── jasmine-reporters @ 0.1.0
├─┬ jquery @ 1.6.3 
│ └── htmlparser @ 1.7.3
├─┬ jsdom @ 0.2.9
│ ├── contextify @ 0.0.6
│ ├── cssom @ 0.2.0
│ ├── htmlparser @ 1.7.3
│ └── request @ 2.2.0
└── underscore @ 1.2.1
